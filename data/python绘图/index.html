matplotlib

在matplotlib种，整个图像为一个figure对象，而在这个figure对象中包含一个或多个axes对象，每个axes对象都是一个拥有自己坐标系的绘图区域。

fig,axes = plt.subplots(nrows, ncols) 接收行列数，返回一个figure对象和axes对象。

axes.set_ylim 设置y轴区间

plt.plot

plt.xticks(roartion=90) 图横坐标标注旋转90度

plt.xlabel()

plt.ylabel()

Plt.title()

plt.figure(figsize=(6,6))定义长宽属性的画布

seaborn

核密度估计

核密度估计（KDE），也叫非参数估计方法，即对数据分布不添加任何假定条件，直接从数据样本本身研究数据分布特征。

seaborn.kdeplot()

而seaborn.distlpot()集成了matplotlib的直方图方法hist() 与kdeplot 方法。

调色板

seaborn.color_palette() 允许传入任何matplotlib制定有效的颜色数组，返回一个可供使用的RGB元祖列表。
color_palette("hls", 8) 使用hls颜色区域定义8种颜色.
color_palette("paired", 8) 定义8种颜色，每两个一对，颜色相近。
coloir_palette("Blues") 还可以传进来一个颜色名称，定义一个渐变色的调色板，默认是8个，渐变是由浅至深，要想变成有深至浅反过来在传参后面加上_r

sns.regplot(x, y, data.....)

热力图

seaborn.heatmap(data, vmin=None, vmax=None, cmap=None, center=None, robust=False, annot=None, fmt='.2g', annot_kws=None, linewidths=0, linecolor='white', cbar=True, cbar_kws=None, cbar_ax=None, square=False, xticklabels='auto', yticklabels='auto', mask=None, ax=None, **kwargs)

重要的参数：data是数据源，可以是一个二维数组。

​ cannot = True 则展示单元格数字

​ cmap 是指色彩颜色的选择

分类柱形图

seaborn.barplot(x=None, y=None, data=None)

箱线图

seaborn.boxplot(x=None, y=None, data=None)

参数data是数据集，x为x轴，y为y轴，如果data是dataframe，x轴和y轴可以是他的列名。

结构化多绘图网格

利用数据集上不同子集的数据在同一张图上绘制子集数据图。

seaborn.FacetGrid(data=None, col=None, row=None, hue=None)

data必须是合适的dataframe数据，col、row、hue都是定义数据自己的变量（列名），每一个都将在网格的不同位置绘制。

但如果仅仅是调用该方法，只会初始化网格，并不会在它们上面绘制任何内容。想要在他身上可视化数据主要用FacetGrid.map()方法。

map方法里要提供为其提供绘图功能的函数方法function，要绘制的数据变量名称args(df列名)，然后会将df那列数据按照调用指定变量的顺序传递给function，还可以通过kwargs传递关键字字典传递该function本身方法的参数。

g.map(function, *args, **kwargs)

“小提琴”图

seaborn.violinplot(x, y, data, hue=None, split=True)
当split为true的时候，画出的小提琴图沿中轴线分别表示hue参数指定的特征两种分类。

联合绘图

seaborn.jointplot(x, y, data, kind)

jointplot方法一般用于两个变量之间的联合绘图，指定x和y的变量，data是数据源，kind参数则是制定绘图的格式。

  1. kind = scatter （默认 ） 带边缘直方图的散点图
  2. kind = reg 带边缘核密度估计的回归图
  3. kind = hex 带边缘直方图的六边形区域图，在数据量大的情况下，可以通过六边形的深浅清晰观测数据的分布（叠合）。
  4. kind = kde kde图
  5. Residents

多变量图

seaborn.pariplot()

此函数将创建一个axes网格，使得传入的data参数重每个变量之间两两组合，形成n*n的对比图。

回归图

seaborn.regplot(data, x, y , x_jitter=.05, ...)

x_jitter 指对x特征的编译量，新特征数据将会在定义范围的偏移量中进行随机的偏移抖动。
